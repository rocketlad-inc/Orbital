CORE IMPLEMENTATION FILES
================================================================================

game_engine.py (498 lines)
  PURPOSE
    - Core game engine that executes ticks atomically
    - Implements the 10-step tick pipeline
    - Manages in-memory game state and persistence

  KEY CLASSES
    - GameEngine: Main orchestrator
    - OrbitalPhysics: Orbital mechanics calculations
    - Database: Abstract database interface
    - GameState: In-memory world snapshot
    - Various data classes (Ship, Body, ManeuverOrder, etc.)

  KEY METHODS
    - execute_tick(): Run one complete tick
    - _load_game_state(): Load world into memory
    - _execute_maneuvers(): Process queued maneuvers
    - _accumulate_production(): Generate resources
    - _advance_research(): Progress technology
    - _advance_production_queue(): Build ships/facilities
    - _save_tick_state(): Persist all changes

  WHEN TO CONSULT
    - You want to understand the tick execution pipeline
    - You need to modify core engine logic
    - You're debugging a specific operation
    - You want to extend the engine

  DEPENDS ON
    - database_adapters.py (Database interface)
    - Standard library (json, logging, dataclasses)

================================================================================

database_adapters.py (496 lines)
  PURPOSE
    - Database abstraction layer
    - Concrete implementations (SQLite, PostgreSQL)
    - Schema initialization
    - Sample data seeding

  KEY CLASSES
    - Database: Abstract interface
    - SQLiteDatabase: Full SQLite implementation
    - PostgreSQLDatabase: PostgreSQL stub
    - DatabaseFactory: Configuration-based creation

  KEY FUNCTIONS
    - init_schema(): Create all tables and indexes
    - seed_sample_game(): Create test data

  WHEN TO CONSULT
    - You need to set up a database
    - You want to use PostgreSQL instead of SQLite
    - You want to implement a custom database
    - You need the schema definition

  DEPENDS ON
    - game_engine.py (Database interface)
    - sqlite3 (for SQLiteDatabase)
