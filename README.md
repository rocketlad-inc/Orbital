# Orbital

Prototype for Orbital — a single-file HTML application.

## Files

- `index.html` — the prototype (open directly in a browser to run)

## Local development

No build step. Open `index.html` in a browser, or serve the folder with any static server:

```sh
python -m http.server 8000
# then visit http://localhost:8000
```

## Versioning

Use git tags for snapshots (e.g., `v3.9`) rather than embedding version numbers in filenames.
